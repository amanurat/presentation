Why SVN ?

  • Powerful with command line interface
  • Free / Open source
  • Faster
  • DB based not file based
  • Customized attributes to be attached to files
  • Automatically handles all file types without instructions
  • Support Folder / File rename & copies maintaining the history

Why GIT ?

  • Distributed - no single of failure
  • Speed - work on local
  • Simple - less config
  • Non-linear development
  • Easy Merging
  • No file locks

Workflow

  • Code
  • Stage your changes (git add)
  • Review your work (git status, git diff)
  • Commit (locally) (git commit)
  • Repeat

## How do we do it ? 

                        arm$ git checkout -b new_feature
                        Switched to a new branch 'new_feature'
                        arm$ git branch
                          master
                        * new_feature
                        arm$ git checkout master
                        Switched to branch 'master'

                          new_feature

## What about merge ? 

                        arm$ git checkout -b update_readme
                        arm$ git branch
                          master
                        * update_readme
                        arm$ vi README.md
                        arm$ git add .
                        arm$ git commit -m "update README"
                        [update_readme 78fb5b8] update README
                         1 file changed, 2 insertions(+)
                        arm$ git status
                        # On branch update_readme
                        nothing to commit (working directory clean)
                        arm$ git checkout master
                        Switched to branch 'master'
                        arm$ git merge update_readme
                        Updating d369bc8..78fb5b8
                        Fast-forward
                         README.md | 2 ++
                         1 file changed, 2 insertions(+)
                        arm$ git status
                        # On branch master
                        nothing to commit (working directory clean)

## Sharing your work 

                        git remote add <remote_name> <remote_url>
                        git push <remote_name> <branch_name>
                        git pull <remote_name> <branch_name>
                        git fetch <remote_name> <branch_name>
## Viewing Changes 

                        git log # show a log of changes
                        git status # see which file you changed
                        git diff # view uncommitted changes in the working copy
                        git show <commitid>	# show one commit
## Undoing local changes 

                        git checkout <filename> #restore original, throw local changes away
                        git checkout -f # throw all local changes away
                        git reset --hard # the same
                        git rm <filename> # remove file

Scenario based: Problem & Solutions

## How to create a local repo ? 

                        git init
## How to clone a remote repo ? 

                        git clone [repo_url]
## How to see history ? 

                        git log --oneline --graph
## How to used alias ? 
git config --global alias.lg "log --graph --decorate --oneline --abbrev-commit"

## How to make a commit ? 

                        git status
                        echo 'hello world' >> file1.txt
                        git status
                        git add file1.txt
                        git status
                        git commit -m "my first commit"

svn add != git add

## Git Diff 

                        echo 'test git diff' >> file1.txt
                        git status
                        git diff
                        git add file1.txt
                        git status
                        # change to be committed # (use "git rm --cached [file]" to unstage)

## Index is a SNAPSHOT!! 

                        echo 'forgot something' >> file1.txt
                        git status
                        git diff
                        git commit -am "add something"

## How to create a new branch ? 

                        git branch feature1
                        git branch
                        git checkout feature1
                        git branch
                        git lg

## How to create a new branch and switch to it ? 

                        git checkout -b feature2
                        git branch
## How to create a new branch from a commit point ? 

                        git checkout -b feature3 HEAD~1
                        git branch
                        git lg --all
## Makes 2 commit to feature2 branch 

                        git checkout feature2
                        echo 'feature 2.1' >> feature2.txt
                        git add feature2.txt
                        git commit -m "completed task 2.1"
                        echo 'feature 2.2' >> feature2.txt
                        git commit -am "complete task 2.2"
                        git lg

## How to merge a branch ? 

                        git checkout master
                        git merge feature2
                        git lg

## Switch back to feature3 branch and create 1 new commit !! 

                        git checkout feature3
                        echo 'feature3' >> feature3.txt
                        git add feature3.txt
                        git commit -m "complete feature3"
                        git lg

## Switch back to master branch and merge feature3 branch !! 

                        git checkout master
                        git merge feature3
                        git lg

## How to reset a branch ? 

                        git checkout master
                        git reset --hard feature2
                        git lg

## How to delete a branch ? 

                        git checkout master
                        git branch -D feature2
                        git branch
                        git lg

## How to rebase a branch ? 

                        git checkout feature3
                        git rebase master
                        git branch
                        git lg

## How to see remote details ? 

                        git remote -v
                        git remote show origin
## How to see tracking remote branches ? 

                        git branch -r
                        git branch -a
## How to push changes to a branch 

                        git push origin master
## How to pull from remote 

                        git pull origin master
                        # git pull == git fetch; git merge origin/master #
                        # git pull --rebase == git fetch; git rebase origin/master #
## How to stash changes ? 

                        git reset --hard HEAD~1
                        echo 'test stash' >> feature2.txt
                        git status
                        git pull origin master
                        git stash
                        git status
                        git pull origin master
                        git stash pop
## How to create a tag ? 

                        git tag v1.0
                        git tag v1.1 [commit]
                        git push origin v1.0
## How to delete a local branch ? 

                        git branch -D feature2
## How to delete a remote branch ? 

                        git push origin :feature2
## How to remove deleted tracking branch ? 

                        git branch -a
                        git fetch
                        git prune
                        git fetch -p
## How to revert a public commit ? 

                        git revert #commit
